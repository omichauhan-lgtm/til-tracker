## Category: Storage Engines
### **Log-Structured Merge-Trees (LSM Trees)**

LSM Trees optimize for write-heavy workloads by writing sequentially to a memory buffer (MemTable). Once full, it flushes to disk as an immutable SSTable (Sorted String Table). Reads query the MemTable and then SSTables via Bloom filters, relying on background Compaction to merge duplicate updates.
