### **Bloom Filters: Space-Efficient Probabilistic Sets**

A Bloom filter is a space-efficient bit array representing a set. It uses multiple hash functions to set bits on insert. Queries check if all bits are set; a negative response guarantees the element is not in the set, while a positive response carries a configurable false-positive probability.
